. O que é Code::Blocks?
- Uma IDE ad hoc para trabalhar com C/C++ e outras linguages.

. O que são compiladores?
- São softwares especiais para converter código em linguagem de máquina.

. O que é iostream?
- Biblioteca C++ para imprimir e receber resposta do console.

. Diferença entre a linguagem C e C++?
- Existem inúmeras diferenças relacionadas a sintaxe e as bibliotecas
usadas pelas duas linguagens. Mas a diferença principal é que a linguagem
C é procedural, enquanto que a linguagem C++ é Multi-paradigma (procedural
e orientada a objetos).
